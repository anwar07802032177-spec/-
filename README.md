# جمعيتي — تطبيق إدارة الجمعيات

تطبيق ويب بلغة Python (Django) لإدارة الجمعيات، بواجهة عربية من اليمين لليسار (RTL).

## ✅ المنجَز حالياً (المرحلة 0 + 1)

- إعداد مشروع Django كامل بالعربية و RTL
- **إدارة الأعضاء:** إضافة / تعديل / حذف / عرض + بحث وفلترة حسب الحالة
- **الاشتراكات السنوية:** ربطها بالأعضاء وتتبّع حالة الدفع
- **لوحة قيادة** بإحصائيات سريعة (عدد الأعضاء، النشطون، الاشتراكات غير المدفوعة)
- **تسجيل دخول** ولوحة إدارة Django جاهزة

## 🚀 التشغيل

```bash
# 1. تفعيل البيئة الافتراضية
source venv/bin/activate

# 2. تطبيق الترحيلات (أول مرة فقط)
python manage.py migrate

# 3. تشغيل الخادم
python manage.py runserver
```

ثم افتح المتصفح على: http://127.0.0.1:8000/

### بيانات الدخول الافتراضية
- المستخدم: `admin`
- كلمة المرور: `admin12345`

> ⚠️ غيّر كلمة المرور قبل النشر الحقيقي.

## 📁 البنية

```
jamaiti/      إعدادات المشروع (عربي + RTL)
```
